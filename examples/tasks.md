# Example tasks

Use these to test whether your planner produces actionable steps.

## Task 1

**Prompt:** "Write a short release note for v1.2.0"

**Expected plan outline:**

- Clarify audience (users vs devs)
- Gather changes (features/fixes)
- Draft release notes with sections
- Check length and completeness

## Task 2

**Prompt:** "Create a checklist for migrating a PostgreSQL database"

**Expected plan outline:**

- Identify scope & constraints
- Plan backup/restore steps
- Plan validation and rollback
- Produce final checklist
